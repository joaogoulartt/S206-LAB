## Lista 2

### Configuração do projeto

Para instalar as dependências do projeto, basta rodar:

```bash
npm install

ou

yarn
```

### Executando os testes

Para executar os testes, basta rodar:

```bash
npx cypress run

ou

yarn test
```

ou

```bash
npx cypress open

ou

yarn open
```

### Gerando relatório

Para gerar o relatório, basta rodar:

```bash
npx cypress run --spec 'cypress/e2e/**/'

ou

yarn report
```
